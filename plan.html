v1.0 Development Roadmap

專案 開發計畫

PromptDaily — 每日自動從 PromptHero 與 OpenArt 蒐集最熱門的 AI 繪圖提示詞，透過 Gmail 寄送到您的信箱。

已確認的決定

  • 個人使用 — 不使用 Firebase，Email 以環境變數管理
  • Gmail SMTP — 使用 Nodemailer + Gmail App Password 寄信
  • GitHub 部署 — 前端 GitHub Pages ＋ 後端 GitHub Actions Cron
  • Midjourney 替代 — 改用 PromptHero 的 Midjourney 分類頁面

系統架構

GitHub Repository ├── index.html ← 訂閱首頁（GitHub Pages 托管） ├── unsubscribe.html ← 取消訂閱頁面 ├── plan.html ← 本頁面（開發計畫） │ ├── scripts/ │ ├── scraper.js ← Puppeteer 爬蟲（抓取 3 來源 × 3 提示詞） │ ├── emailTemplate.js← HTML 信件模板產生器 │ └── mailer.js ← Nodemailer SMTP 寄信模組 │ └── .github/workflows/ └── daily-prompt.yml← 排程：每天 21:00 UTC（台灣 05:00） ├─ Step 1: 執行爬蟲抓取提示詞 ├─ Step 2: 組裝 HTML 信件 └─ Step 3: Gmail SMTP 寄出信件
